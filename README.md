# Cosmic Death Fungus — NAC Protocol Documentation

Community documentation for the NAC Candida/Fungal Protocol, originally authored by **Peribsen & MacLir**.

> **Not medical advice.** This is community-documented information. Consult a qualified healthcare professional before beginning any protocol.

## What Is This?

This repository is the canonical, search-engine-indexed, version-controlled source for the CDF NAC Protocol — a community-developed approach to addressing systemic fungal infection. The GitHub Pages site at [cosmicdeathfungus.github.io](https://cosmicdeathfungus.github.io) is the primary reference, built to survive platform restrictions and ensure permanent public access.
